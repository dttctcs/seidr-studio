## Usage

Seidr Studio provides a modern user interface to visualize your seidr api capabilities

### Development

- Install dependencies: `npm i`
- Run: `npm start`

### Production

- Install dependencies: `npm i`
- Run `npm build` (see also https://create-react-app.dev/docs/production-build/)
